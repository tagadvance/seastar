PLAN: Support DROP KEYSPACE [IF EXISTS] (DropKeyspaceHandler)

Low effort. Mirrors CreateKeyspaceHandler in reverse.

READ FIRST
- Skill: .claude/skills/add-cql-statement/SKILL.md
- CreateKeyspaceHandler.java
- VolatileDriverContext.java -- confirm there is a remove-keyspace method; if only newSeaStarKeyspace
  and getSeaStarKeyspace exist, add removeSeaStarKeyspace(CqlIdentifier) to the context interface +
  VolatileDriverContext (remove from the keyspace map under lock).

INSPECT
  ./gradlew :lib:inspectRaw -Pquery="DROP KEYSPACE foo" --console=plain -q 2>/dev/null
  ./gradlew :lib:inspectRaw -Pquery="DROP KEYSPACE IF EXISTS foo" --console=plain -q 2>/dev/null

Raw class: org.apache.cassandra.cql3.statements.schema.DropKeyspaceStatement$Raw (confirm)
Fields: keyspace name (String or QualifiedName), boolean ifExists.

IMPLEMENTATION
1. New handler DropKeyspaceHandler implements CqlHandler for the DropKeyspaceStatement raw type.
2. If keyspace absent:
     - ifExists -> succeed.
     - else -> InvalidQueryException (verify exact message/type via container).
   Else context.removeSeaStarKeyspace(name). If the dropped keyspace is the session's current
   keyspace, clear it (check how session keyspace is held; UseKeyspaceHandler sets it via a Consumer).
3. Register in the CqlHandlerRegistry list in SeaStarCqlRequestHandler.

FIDELITY (verify against ContainerCqlSessionTest)
  - missing keyspace, no IF EXISTS -> InvalidQueryException.

TESTS (AbstractCqlSessionTest)
  Create keyspace, DROP it, assert getSeaStarKeyspace empty. DROP again without IF EXISTS -> throws.
  DROP IF EXISTS on missing keyspace -> no throw.
  (Order this test LAST or use a throwaway keyspace name so it doesn't delete "foo" that earlier
   ordered tests depend on.)

VERIFY
  ./gradlew :lib:test --tests 'com.tagadvance.seastar.ContainerCqlSessionTest'
  ./gradlew :lib:test --tests 'com.tagadvance.seastar.SeaStarCqlSessionTest'
