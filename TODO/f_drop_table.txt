PLAN: Support DROP TABLE [IF EXISTS] (DropTableHandler)

Low effort. Storage support already exists: VolatileKeyspace.removeSeaStarTable(CqlIdentifier).

READ FIRST
- Skill: .claude/skills/add-cql-statement/SKILL.md
- CreateTableHandler.java (keyspace resolution pattern), VolatileKeyspace.removeSeaStarTable.

INSPECT
  ./gradlew :lib:inspectRaw -Pquery="DROP TABLE foo.users" --console=plain -q 2>/dev/null
  ./gradlew :lib:inspectRaw -Pquery="DROP TABLE IF EXISTS foo.users" --console=plain -q 2>/dev/null

Raw class: org.apache.cassandra.cql3.statements.schema.DropTableStatement$Raw
Fields: QualifiedName name (keyspace may be null -> use session keyspace); boolean ifExists.

IMPLEMENTATION
1. New handler DropTableHandler implements CqlHandler<DropTableStatement.Raw>.
2. Resolve keyspace (name.keyspace or session keyspace). InvalidQueryException if no keyspace.
3. If table absent:
     - ifExists -> succeed (empty result).
     - else -> InvalidQueryException (confirm message via container; Cassandra says
       "Table '<ks>.<t>' doesn't exist" -- verify exact type/text).
   Else keyspace.removeSeaStarTable(name); also call the table's drop() to detach if appropriate.
4. Register in the CqlHandlerRegistry list in SeaStarCqlRequestHandler.

FIDELITY (verify against ContainerCqlSessionTest)
  - missing table, no IF EXISTS -> InvalidQueryException.
  - missing keyspace -> InvalidQueryException.

TESTS (AbstractCqlSessionTest)
  Create table, DROP it, assert getSeaStarTable is empty. DROP again without IF EXISTS -> throws.
  DROP IF EXISTS on a missing table -> does not throw.

VERIFY
  ./gradlew :lib:test --tests 'com.tagadvance.seastar.ContainerCqlSessionTest'
  ./gradlew :lib:test --tests 'com.tagadvance.seastar.SeaStarCqlSessionTest'
